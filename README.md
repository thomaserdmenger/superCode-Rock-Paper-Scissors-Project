# SuperCode Rock Paper Scissors Project

## Purpose of the website

Play rock, paper, scissors against the computer. The first player to reach 5 points is the overall winner. Restart the game to play again.

## Built with

- Responsive Design
- Flexbox Layout
- JavaScript ES6+
